# 🩸 Bloodshed Mod Menu

A BepInEx plugin for the Steam game **[Bloodshed](https://store.steampowered.com/app/2747550/Bloodshed/)**.

---

## ✨ Features

| Feature | Description |
|---|---|
| **Aimbot** | Automatically aims at the nearest alive enemy within a configurable radius. Smoothing and radius are adjustable via in-game sliders. |
| **Immortal Mode** | Locks your HP at 100. |
| **Enemy Counter** | Displays a live count of all alive enemies in the current level. |
| **Toggle Menu** | Press `M` to open or close the mod menu at any time. |

---

## 📋 Requirements

- **[Bloodshed](https://store.steampowered.com/app/2747550/Bloodshed/)**
- **BepInEx (IL2CPP)** — [Download here](https://builds.bepinex.dev/projects/bepinex_be) Download BepInEx-Unity.IL2CPP-win-x64-6.0.0-be.755+3fab71a.zip

---

## 🚀 Installation

1. Install **BepInEx (IL2CPP)** into your Bloodshed game folder. Follow the [BepInEx installation guide](https://docs.bepinex.dev/articles/user_guide/installation/index.html) if you haven't done this before.

2. Download the latest **`mod_menu.dll`** from the [Releases](https://github.com/Wambo420Rambo/Bloodshed_Mod_Menu/releases) 

3. Place `mod_menu.dll` into:
   ```
   <Bloodshed game folder>/BepInEx/plugins/
   ```

4. Launch **Bloodshed**

---

## 🎮 How to Use

- Press **`M`** in-game to open or close the mod menu.
- Use the toggle buttons to enable or disable each cheat.
- Adjust the **Smoothing** and **Radius** sliders to fine-tune the aimbot.
